INTRO TO MAVEN & GEOJSON
- add new dependencies like Geo-JSON to pom.xml
- need artefactid, version, groupid and scope from web
- Geo-JSON:
	- FeatureCollection contains Features, every Feature has a Geometry, every Geometry has Coordinates
	- Geometries - Point, LineString, Polygon
	- LineString has an array of Coordinates of 2 or more positions
	- Coordinates are (Longitude, Latitude) - positive latitude, negative longitude
- Show map on http://geojson.io, paste map code in
- build up map by creating lots of features
- new stuff made by static methods:
	- Point.fromLngLat(lng, lat) makes a point at a long,lat - lng and lat are doubles
	- LineString.fromLngLats(List(Point)) makes a line from an array of coordinates
	- Polygon.fromLngLats(List.of(List(Point))) makes a polygon
- (Geometry)point-name to get geometry
- casting:
	- Feature.fromGeometry(g)
	- FeatureCollection.fromFeatures(List(Feature))
- Feature.addStringProperty("a", "b") maps a to b, Feature.addNumberProperty("a", 10)
- FeatureCollection.toJson()
